# ☁️ Cloud WAF (Web Application Firewall) Simulation

An interactive dashboard that simulates a cloud-based Web Application Firewall.  
Demonstrates real-time threat detection, rule configuration, and security monitoring.

## Features
- Toggle WAF rules (SQLi, XSS, Path Traversal, Bad Bots)
- Simulate HTTP requests (normal or malicious)
- Live blocking decisions with explanation
- Security score and event logging
- Ready for internship portfolio presentations

## How it works
1. Open `index.html` in any modern browser.
2. Enable/disable WAF rules.
3. Click preset attack buttons or type custom URLs.
4. See blocked requests appear in logs with reason.

## Technology
- HTML5, CSS3, JavaScript
- No backend — purely frontend simulation
- Regex-based pattern matching for OWASP top threats

## Use case
Perfect for cloud security demos, internship projects, or understanding how WAF protects web applications.
